🏢
Property Manager Wizard helps you track your entire portfolio — tenants, rent, maintenance, and financials — from one clean dashboard. Everything stored locally, accessible anytime.
🚀 Getting started
1
Add each property from the Portfolio tab. Include address, bedrooms, and expected rent.
2
Add tenants in the Tenants tab. Link each tenant to a property with their lease dates.
3
Use Rent to mark each payment as received month by month.
4
Log all maintenance requests with priority — track from open → in progress → done.
5
Add expenses in Financials and download your IRD-ready report at tax time.
📊 Features
🏠
Portfolio Dashboard
All properties at a glance — status, occupancy, weekly rent, and quick actions.
👥
Tenant Management
Lease progress bars, contact details, and lease expiry alerts.
💰
Rent Tracker
Mark payments received month by month. See arrears instantly.
🔧
Maintenance Workflow
Prioritised task list from open to done. Cost tracking per job.
📊
Financial Reports
Month-by-month cash flow, expenses by category, IRD-ready export.
⬇
CSV Export
Download full income/expense report for accountant or IRD.
